# Document Intelligence RAG System

A Retrieval-Augmented Generation (RAG) system that enables semantic search and question answering over documents using transformer embeddings and vector similarity search.

## Features
- Upload and index PDF documents
- Automatic text extraction and document chunking
- Transformer embeddings using Sentence Transformers (MiniLM)
- Vector similarity search for retrieving relevant document sections
- Question answering using an LLM
- FastAPI backend with interactive API documentation

## Tech Stack
- Python
- FastAPI
- Sentence Transformers (all-MiniLM-L6-v2)
- FAISS / Vector Similarity Search
- Google Gemini API
